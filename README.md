# Face-Detection-Streamlit-App

A computer vision web application built with Streamlit and OpenCV.

## Features
- Face detection
- Eye detection
- Webcam image capture
- Multi-page Streamlit app
- Adjustable detection strictness

## Technologies Used
- Python
- Streamlit
- OpenCV
- NumPy
- Pillow

## Installation

```bash
pip install -r requirements.txt
```

## Run the App

```bash
streamlit run app.py
```
